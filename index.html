SOBRE O PCC

Como funciona o site:

Este site permite que você codifique e decodifique textos usando a criptografia PCC. Basta digitar o texto no primeiro campo e usar os botões para codificar ou decodificar. O resultado aparece logo abaixo.

O que é PCC?

PCC significa PUDDING CODE CHARACTERS. É um sistema de criptografia criado por Guylherme (13 anos), que utiliza caracteres especiais para transformar letras e números, criando uma forma divertida e única de codificar mensagens. O nome “PUDDING” foi escolhido porque eu gosto de pudim!

O que cada botão faz:

  • Codificar: transforma seu texto normal em texto criptografado usando o PCC.
  • Decodificar: transforma um texto criptografado de volta ao formato original.
  • Copiar: copia o resultado da decodificação para a área de transferência.
  • Limpar tudo: apaga o conteúdo dos dois campos de texto.
  • Modo escuro: alterna entre o modo claro e escuro para melhorar a leitura.

Importante sobre o uso:

O sistema PCC foi criado para funcionar com letras simples (alfabeto latino) e alguns caracteres acentuados. Certos símbolos ou emojis podem não ser compatíveis. Recomendamos utilizar apenas letras (a-z), números e sinais de pontuação básica. Estamos sempre melhorando a compatibilidade!

Se algo der errado:

Se algum botão não funcionar, tente:

  • Atualizar a página.
  • Verificar se o navegador tem permissão para copiar/colar (especialmente no botão "Copiar").
  • Usar um navegador moderno como o Google Chrome.

Se o problema continuar, você pode relatar no meu e-mail.
